## ğŸ“¥ Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/marwa-q/Laravel-crud.git
   ```

   ```sh
   cd Laravel-crud
   ```
3. Install dependencies:
   ```sh
   composer install
   ```

   ```sh
   cp .env.example .env
   ```

   ```sh
   php artisan key:generate
   ```

   ```sh
   php artisan migrate
   ```
7. Start the development server:
   ```sh
   php artisan serve
   ```

   ```
   http://127.0.0.1:8000
   ```
